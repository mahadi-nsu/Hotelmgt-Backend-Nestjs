## Description

Simple Club management User endpoints(GET , POST , PATCH , DELETE)

## Installation (clone the repo and enter into the project folder)

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```
